## [0.6.0] — 2026-02-24

### ✨ 추가
- **종교개혁 확장판 (Reformation)** — 새 게임 모드 선택, 진영(충성파/개혁파) 시스템, 최대 10인 지원
- **인퀴지터 (Inquisitor)** — 대사 대체 캐릭터, 심문(상대 카드 확인 후 교체 강제) + 교환(1장) 능력
- **전향 (Conversion)** — 자기 전향(1코인) / 타인 전향(2코인)으로 진영 변경, 코인은 재무부로
- **횡령 (Embezzlement)** — 공작 능력, 재무부 코인 전액 획득
- **심문 (Examine)** — 인퀴지터 능력, 상대 비공개 카드 확인 후 돌려주기/교체 선택 (`ExamineModal`)
- **재무부 (Treasury)** — 전향 비용이 재무부에 누적, 횡령으로 회수 가능
- **진영 제한** — 같은 진영 공격 불가 (전원 같은 진영이면 해제), 해외원조 블록도 진영 제한 적용
- **방 목록 진행중 게임 표시** — 로비에서 대기 중(초록) / 게임 중(주황) 배지, 생존자 수 표시
- **ExchangeModal 카드 swap 선택 UX** — 드래그 방식 대신 카드 탭 순서 배지로 직관적 교체
- **exchange_select 45초 타임아웃** — 시간 초과 시 자동 기존 카드 유지
- **AlertModal 통합** — `window.alert`/`window.confirm`을 커스텀 모달로 전면 교체
- **닉네임 없는 URL 접속 리다이렉트** — 닉네임 미입력 상태로 게임 URL 직접 접속 시 로비로 이동
- **미니 로그 타임스탬프 정렬** — structuredLog 기반 정확한 시간순 정렬 + private 로그 필터링
- **타이머 바 자동 행동 안내** — "시간 초과 시 자동 소득" / "자동 쿠데타" 텍스트 표시

### 🔧 개선
- **인원수 기반 덱 조정** — reformation 6인 이하 3장, 8인 이하 4장, 9인 이상 5장씩 생성
- **방 목록 라벨** — "대기 중인 방" → "방 목록" (진행중 방 포함으로 변경)
- **maxPlayers 동적 조정** — reformation 모드 10명, 기본 6명

### 🐛 버그 수정
- 퇴장/추방 시 `window.alert` 대신 AlertModal로 교체하여 모바일 호환성 개선
- 미니 로그에 다른 플레이어의 private 로그가 노출되던 문제 수정

## [0.4.0] — 2025-02-24

### ✨ 추가
- **게임 규칙 모달** — 게임 중 📖 버튼으로 언제든 규칙 확인 (`GameRulesModal`)
- **404 / 에러 페이지** — 잘못된 URL, 없는 방 코드, 런타임 에러 전용 폴백 페이지 (`not-found.tsx`, `error.tsx`)
- **방 없음 안내 UI** — 존재하지 않는 방 코드 입력 시 입력된 코드와 안내 메시지 표시
- **추방 재접속 차단** — kick 시 `kickedPlayerIds` 배열에 ID 기록, join 시 403 반환
- **랜덤 선공** — 매 게임 시작 시 첫 번째 플레이어 무작위 선택
- **턴 구분선 로그** — `--- {이름}의 턴 ---` 형식의 구분선으로 로그 가독성 향상
- **구조화된 게임 로그** — `LogEntry` 기반 아이콘 + 컬러 + 턴 구분선 (`StructuredLogEntry`)
- **암살 도전 경고 배너** — 도전 실패 시 2명 피해 안내 빨간 배너 + 예고 로그
- **채팅 로그 통합** — 퀵챗 메시지가 게임 로그에 💬 표시로 병합
- **낙관적 채팅 UI** — 내 채팅 즉시 표시, 상대 메시지는 Firebase 구독으로 수신
- **게임 오버 대기 UI** — 비방장 플레이어에게 "방장의 재시작을 기다리는 중..." 표시

### 🔧 개선
- **용어 통일** — "블러프" → "거짓말", "쿠" → "쿠데타", "세금 걷음" → "징수", "강탈" → "갈취"
- **암살 도전 버튼** — "도전! (암살자가 아니라고 생각해요)" 전용 문구 추가
- **ResponseModal 모바일** — 패딩 반응형 (`px-4 sm:px-6`), 버튼 폰트 크기 조절 (`text-xs sm:text-sm`)
- **렌더링 최적화** — `WaitingResponseIndicator`, `QuickChat`, `ChatBubble`에 `React.memo()` 적용
- **타임아웃 fetch 중복 제거** — `fireTimeout` useCallback 헬퍼로 ~40줄 절감
- **QuickChat 쿨다운** — `useState` → `useRef`로 변경하여 리렌더 방지
- **Firebase 채팅 구독** — `onValue` → `onChildAdded` + `query(orderByChild, startAt)` 으로 신뢰성 향상
- **채팅 메시지 자동 정리** — 60초 경과 메시지 자동 삭제 (30초 쓰로틀)

### 🐛 버그 수정
- 추방된 플레이어가 같은 방에 재입장 가능했던 문제 수정
- 다른 유저의 채팅 메시지가 표시되지 않던 문제 수정
- 추방 후 `clearActiveRoom()` 누락으로 자동 재접속 루프 발생하던 문제 수정
